An interactive **Tableau dashboard** that analyzes BMW's global sales performance over 15 years. This project uncovers key insights on fuel-type balance, regional growth (especially Asia), model performance, and long-term sales trends.

**Live Demo / Screenshots**: https://github.com/deephaldankar1135/BMW-Global-Sales-Performance-Dashboard-2010-2024-/blob/main/Screenshot%202025-11-30%20200947.png

## 🎯 Business Problem & Objective
BMW is transitioning toward electrification while maintaining strong performance in traditional powertrains. This dashboard helps stakeholders answer:
- How balanced is the powertrain portfolio (Petrol, Diesel, Electric, Hybrid)?
- Which region drives growth?
- Which models deliver the highest volume?
- How has sales recovered post-2020?

## 📊 Key Insights
- **Balanced Powertrain Mix**: Almost perfect ~25% share across Petrol, Diesel, Electric, and Hybrid – showing successful diversification.
- **Asia Dominates**: Highest sales volume (~40M units) and identified as the best-performing region.
- **Top Performers**: 7 Series leads in flagship sales; 3 Series remains the volume king.
- **Strong Recovery**: V-shaped growth post-2020, ending above pre-pandemic levels.
- **Portfolio Strength**: Sedans (3/5/7 Series) + X-line SUVs contribute ~70% of volume.

**Strategic Recommendation**: Accelerate X-line electrification in Asia to capture SUV demand while leveraging the balanced powertrain strategy.
